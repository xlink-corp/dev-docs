# 能源基础管理用户手册

> 版本：v1.0 | 更新日期：2026-01

## 一、模块概述

### 1.1 什么是能源基础管理？

能源基础管理是构建能源管理体系的基础数据配置模块，为能耗采集、分析、统计提供数据支撑。通过本模块可以：

- **定义能源类型**：配置组织使用的能源种类及计量参数
- **管理计量表**：维护能源计量设备及其数据采集策略
- **定义用能对象**：建立用能实体与计量表的映射关系
- **配置能源源头**：定义能源输入来源，支撑能源平衡分析
- **配置能流结构**：可视化配置能源流向拓扑
- **管理基线**：建立能效基准线，支持 ISO 50006/50015 标准
- **配置能源参数**：电价策略、峰谷时段、班次日历等

### 1.2 核心子模块

| 子模块 | 功能说明 | 入口路径 |
|--------|----------|----------|
| 能源类型管理 | 定义能源种类及计量参数 | 能源管理 → 能源类型管理 |
| 计量表管理 | 管理计量设备及采集策略 | 能源管理 → 计量表管理 |
| 用能对象管理 | 定义用能实体并绑定表计 | 能源管理 → 用能对象管理 |
| 能源源头管理 | 定义能源输入来源 | 能源管理 → 能源源头 |
| 能流配置 | 配置能源流向拓扑结构 | 能源管理 → 能流配置 |
| 基线管理 | 配置能效基线 | 能源管理 → 基线管理 |
| 能源配置设置 | 电价、时段、班次等配置 | 能源管理 → 能源配置设置 |

### 1.3 用户角色与权限

| 角色 | 主要职责 | 常用功能 |
|------|----------|----------|
| 能源管理员 | 建立能源基础数据体系 | 全部功能 |
| 设备管理员 | 维护计量表信息 | 计量表管理、采集策略 |
| 运维人员 | 监控采集状态 | 采集监控、数据质量 |

## 二、快速入门

### 场景 1：首次建立能源管理体系

**目标**：从零开始配置组织的能源管理基础数据

**步骤**：

1. **启用能源类型**
   - 进入「能源管理 → 能源类型管理」
   - 查看系统预设的能源类型（电力、天然气、水等）
   - 点击切换启用需要的能源类型
   - 如需自定义参数，点击「覆盖修改」调整折标煤系数等

2. **创建计量表**
   - 进入「能源管理 → 计量表管理」
   - 点击「添加计量表」
   - 填写基本信息：名称、编码、能源类型、位置
   - 关联 IoT 设备
   - 选择或创建采集策略

3. **定义用能对象**
   - 进入「能源管理 → 用能对象管理」
   - 点击「添加用能对象」
   - 选择类型（设备/区域/产线等）
   - 绑定对应的计量表

4. **配置电价策略**
   - 进入「能源管理 → 能源配置设置」
   - 配置电价时段（峰、谷、平、尖峰）
   - 配置各时段电价

5. **可选：配置能流拓扑**
   - 进入「能源管理 → 能流配置」
   - 创建能流节点和路径
   - 用于能源平衡分析

---

### 场景 2：添加新计量表并启用自动采集

**目标**：新增一块电表，实现自动数据采集

**步骤**：

1. 进入「计量表管理」→ 点击「添加计量表」

2. 填写基本信息：
   ```
   计量表名称：1#车间总电表
   计量表编码：EM-WS01-001
   能源类型：电力
   安装位置：1#车间配电室
   ```

3. 关联 IoT 设备：
   ```
   设备 ID：填写 Xlink 平台的设备标识
   ```

4. 配置采集策略：
   - 选择已有策略，或点击「新建策略」
   - 设置读数属性标识（如 `totalEnergy`）
   - 设置采集间隔（如 15 分钟）

5. 保存后，系统将按策略自动采集数据

---

### 场景 3：为区域配置能耗统计

**目标**：统计某个车间的整体能耗

**步骤**：

1. 进入「用能对象管理」→ 点击「添加用能对象」

2. 选择对象类型「用能区域」

3. 填写信息并关联空间：
   ```
   对象名称：1#车间
   关联空间：选择对应的空间位置
   ```

4. 绑定计量表：
   - 添加该区域内的所有计量表
   - 如有共用表，设置分成比例（如 70%）

5. 保存后，能耗报表中即可按区域统计

## 三、功能模块详解

### 3.1 能源类型管理

#### 功能概述

定义组织使用的能源种类及其计量参数，是能源管理的基础配置。

#### 页面布局

- **顶部**：Tab 切换（系统预设类型 / 自定义类型）
- **筛选区**：搜索框、能源大类筛选
- **类型列表**：卡片式展示各能源类型

#### 核心功能

| 功能 | 说明 |
|------|------|
| 系统预设类型 | 电力、天然气、蒸汽、水等常用能源，不可删除 |
| 启用/禁用 | 切换开关控制是否在系统中可用 |
| 自定义类型 | 添加特殊能源类型（如压缩空气、氢气） |
| 参数配置 | 折标煤系数、CO₂排放因子、kWh换算因子 |
| 覆盖修改 | 修改系统类型的参数值，可恢复默认 |

#### 关键字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 能源编码 | 唯一标识符 | electricity、natural_gas |
| 能源名称 | 显示名称 | 电力、天然气 |
| 能源大类 | 分类标签 | 化石能源、电力、可再生能源 |
| 来源模式 | 外购/自产/混合 | purchased、self_produced |
| 计量单位 | 标准计量单位 | kWh、m³、吨 |
| 折标煤系数 | 转换为标准煤的系数 | 电力：0.1229 kgce/kWh |
| CO₂排放因子 | 单位能源的碳排放量 | 电力：0.5810 kgCO₂/kWh |
| kWh换算因子 | 转换为 kWh 的系数 | 天然气：9.78 kWh/m³ |

#### 操作指南

**启用能源类型**：
1. 在系统类型列表中找到目标类型
2. 点击右上角的启用开关
3. 启用后该类型可在计量表、用能对象等处选择

**覆盖修改参数**：
1. 点击能源类型卡片进入详情
2. 点击「保存更改」按钮
3. 修改所需参数值
4. 保存后该组织使用修改后的参数

**添加自定义类型**：
1. 切换到「自定义类型」Tab
2. 点击「添加能源类型」
3. 填写必填信息：编码、名称、单位、大类
4. 填写可选参数：折标煤系数、排放因子等
5. 保存并启用

### 3.2 计量表管理

#### 功能概述

管理能源计量设备，关联 IoT 设备实现自动数据采集。

#### 页面布局

- **统计卡片**：总计量表数、在线、异常、离线
- **筛选区**：搜索框、能源类型筛选、状态筛选
- **操作区**：添加计量表、采集策略、批量操作
- **列表区**：计量表信息列表

#### 计量表状态说明

| 状态 | 说明 | 视觉标识 |
|------|------|----------|
| 在线 | 设备正常通信，数据正常采集 | 🟢 绿色圆点 |
| 异常 | 数据异常或通信不稳定 | 🟡 黄色圆点 |
| 离线 | 设备无法通信 | ⚫ 灰色圆点 |

#### 列表字段说明

| 字段 | 说明 |
|------|------|
| 计量表名称/编号 | 名称和唯一编码 |
| 能源类型 | 电力、天然气等 |
| 位置 | 安装空间位置 |
| 关联设备 | Xlink IoT 设备信息 |
| 采集策略 | 当前绑定的采集策略 |
| 最后读数时间 | 最近一次数据采集时间 |
| 操作 | 编辑、查看读数、删除 |

#### 关键操作

**添加计量表**：
1. 点击「添加计量表」按钮
2. 填写基本信息：
   - 计量表名称（必填）
   - 计量表编码（必填，唯一）
   - 能源类型（必填）
   - 计量表类型（必填）
   - 所在位置（选择空间）
3. 关联 IoT 设备（可选）：
   - 选择产品并选择设备 
4. 选择采集策略（可选）：
   - 从已有策略中选择
   - 或稍后在采集策略中配置
5. 保存

**查看读数/能耗**：
1. 点击列表行或「查看读数」按钮
2. 右侧滑出抽屉面板
3. 查看历史读数曲线和能耗数据
4. 可切换时间范围

**批量导入**：
1. 点击「批量导入」
2. 下载 Excel 模板
3. 按模板格式填写计量表信息
4. 上传文件，预览并确认导入

### 3.3 采集策略管理

#### 入口

计量表管理页面 → 点击「采集策略」按钮

#### 功能概述

定义自动数据采集的规则，关联到计量表后实现自动同步 IoT 设备数据。

#### 四大功能 Tab

| Tab | 功能说明 |
|-----|----------|
| 采集策略 | 策略的增删改查，定义采集规则 |
| 采集监控 | 查看采集任务执行状态，支持手动触发同步 |
| 能耗处理监控 | 监控读数→能耗的处理任务状态 |
| 数据质量 | 数据完整性、异常检测仪表盘 |

#### 采集策略配置

| 字段 | 说明 | 示例 |
|------|------|------|
| 策略名称 | 策略标识名称 | 电表15分钟采集 |
| 能源类型 | 适用的能源类型 | 电力 |
| 读数属性标识 | IoT 设备上报的属性名 | totalEnergy |
| 采集间隔 | 数据采集频率 | 5分钟 / 15分钟 / 1小时 / 12小时 |
| 启用状态 | 是否启用该策略 | 开关控制 |

#### 操作指南

**创建采集策略**：
1. 在「采集策略」Tab 点击「添加策略」
2. 填写策略名称和能源类型
3. 配置读数属性标识（与 IoT 设备属性一致）
4. 选择采集间隔
5. 保存并启用

**关联计量表**：
1. 进入计量表编辑页面
2. 在「采集策略」字段选择目标策略
3. 保存后立即生效

**手动同步数据**：
1. 切换到「采集监控」Tab
2. 找到目标任务，点击「立即同步」
3. 查看同步结果

### 3.4 用能对象管理

#### 功能概述

定义用能实体（设备、区域、产线等），建立计量表与业务对象的映射关系，实现按对象维度的能耗统计。

#### 用能对象类型

| 类型 | 说明 | 典型场景 |
|------|------|----------|
| 设备-单个资产设备 | 单个用能设备 | 空调、电机、空压机、注塑机 |
| 产线-生产线/工序 | 生产线整体 | 产线能耗统计、产品能耗分摊 |
| 空间 | 空间区域 | 车间、办公区、仓库 |
| 系统-能源系统 | 能源子系统 | 暖通系统、照明系统、动力系统 |

#### 页面布局

- **Tab 切换**：按对象类型分 Tab 显示
- **筛选区**：搜索、状态筛选
- **操作区**：添加、批量操作、导出
- **列表区**：用能对象信息列表

#### 表计绑定规则

| 规则 | 说明 |
|------|------|
| 多表绑定 | 一个用能对象可绑定多个计量表 |
| 分成比例 | 共用表可设置归属比例（如 70%） |
| 生效时间 | 支持设置绑定关系的生效时间区间 |

#### 操作指南

**添加用能对象**：
1. 点击「添加用能对象」
2. 选择对象类型（设备/区域/产线/系统等）
3. 填写基本信息：
   - 对象名称（必填）
   - 对象编码（必填，唯一）
   - 关联实体（设备→资产，区域→空间，产线→生产线，系统→系统类型）
4. 绑定计量表：
   - 点击「添加表计」
   - 选择计量表
   - 设置分成比例（默认 100%）
   - 设置生效/失效时间（可选）
5. 保存

**批量操作**：
- 批量启用/停用：选中多个对象后操作
- 批量删除：选中后点击删除
- 导出 Excel/CSV：导出当前列表数据

#### 关键字段

| 字段 | 说明 |
|------|------|
| 源头名称 | 显示名称 |
| 源头编码 | 唯一标识 |
| 源头类型 | 电网/自发电/可再生/化石燃料等 |
| 能源类型 | 关联的能源类型 |
| 描述 | 详细说明 |
| 启用状态 | 是否启用 |
| 关联节点 | 被多少能流节点引用 |

#### 操作指南

**添加能源源头**：
1. 点击「添加能源源头」

### 3.6 能流配置

#### 功能概述

配置能源流向拓扑结构，可视化展示能源从源头到终端的流动路径，支撑能源平衡计算。

#### 三大功能 Tab

| Tab | 功能说明 |
|-----|----------|
| 拓扑结构 | 可视化能流网络图，展示节点和路径 |
| 节点管理 | 定义能流节点（输入/输出/转换点） |
| 路径管理 | 定义节点间的能流连接关系 |

#### 节点类型

| 类型 | 说明 |
|------|------|
| 输入节点 | 能源进入点，关联能源源头 |
| 消费节点 | 能源消耗点，关联用能对象 |
| 转换节点 | 能源转换点，如变压器 |
| 分配节点 | 能源分配点 |

#### 拓扑图操作

- **拖拽**：调整节点位置
- **点击节点**：查看节点详情
- **点击边**：查看路径详情
- **缩放**：鼠标滚轮或按钮控制
- **平移**：拖拽画布空白区域

### 3.7 基线管理

#### 功能概述

建立能效基准线，用于能效评估和持续改进，遵循 ISO 50006 / ISO 50015 标准。

#### 基线类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 固定基线 | 基于历史固定时段的平均值 | 简单场景，产量稳定 |
| 滚动基线 | 基于最近 N 期的移动平均 | 产量波动，持续改进 |
| 回归基线 | 基于多变量回归模型 | 复杂场景，多因素影响 |

#### 时间粒度

- 日基线
- 周基线
- 月基线

#### 操作指南

**创建基线**：
1. 点击「新建基线」
2. 选择关联的 KPI 定义
3. 选择基线类型
4. 设置时间粒度
5. 配置参数（如回归变量）
6. 设置生效时间
7. 保存

**运行基线计算**：
1. 选择目标基线
2. 点击「运行计算」
3. 系统根据历史数据计算基线值
4. 查看计算结果

**查看趋势**：
1. 进入基线详情
2. 查看历史基线值与实际值对比图表
3. 分析偏差趋势

### 3.8 能源配置设置

#### 入口

能源管理 → 能源价格配置

#### 功能概述

配置能源管理所需的各类参数，包括峰谷平时段、电价配置、能源价格等。

---

#### 3.8.1 峰谷平时段配置

配置一天内的峰谷时段划分。

| 时段类型 | 说明 | 颜色标识 |
|----------|------|----------|
| 谷段 (valley) | 用电低谷期，电价最低 | 🟢 绿色 |
| 平段 (flat) | 正常用电期 | ⚪ 灰色 |
| 峰段 (peak) | 用电高峰期，电价较高 | 🟡 黄色 |
| 尖峰 (sharp) | 极端高峰期，电价最高 | 🔴 红色 |

**可视化时间轴**：
- 24 小时时间轴直观展示各时段分布
- 支持跨午夜时段（如 23:00 - 06:00）

**操作指南**：
1. 点击「添加时段」
2. 选择时段类型
3. 设置开始时间和结束时间
4. 保存

---

#### 3.8.2 电价配置

配置各峰谷时段的电价。

| 字段 | 说明 |
|------|------|
| 方案名称 | 如"2024年电价"、"夏季电价" |
| 时段类型 | 对应的峰谷时段 |
| 单价（元/kWh） | 该时段的电价 |
| 生效日期 | 价格生效起始日期 |
| 失效日期 | 可选，价格失效日期 |
| 电压等级 | 可选，10kV/35kV/110kV 等 |

**操作指南**：
1. 点击「新建电价」
2. 填写价格名称
3. 选择时段类型
4. 输入电价
5. 设置生效日期
6. 可选：设置失效日期和电压等级
7. 保存

---

#### 3.8.3 能源价格配置

配置天然气、蒸汽、水等非电能源的价格。

**计价方式**：

| 方式 | 说明 |
|------|------|
| 单一价格 | 固定单价，不随用量变化 |
| 阶梯计价 | 按用量分段定价，用得越多单价可能越高 |
| 分时计价 | 按时段定价，类似电力峰谷 |

**操作指南**：
1. 选择能源类型
2. 选择计价方式
3. 配置价格参数
4. 设置生效时间
5. 保存

### 3.9 能源配置设置

#### 入口

生产管理 → 生产基础配置

#### 功能概述

配置生产管理所需的各类参数，包括产品目录、生产单元、班次日历等。

#### 3.9.1 产品目录

用于产品能耗核算和碳足迹分析。

| 字段 | 说明 |
|------|------|
| 产品编码 | 唯一标识 |
| 产品名称 | 产品名称 |
| 计量单位 | 件、吨、米等 |
| 标准能耗 | 单位产品的标准能耗 |
| 标准工时 | 单位产品的标准工时 |
| 所属产线 | 关联的生产线 |

---

#### 3.9.2 生产单元-生产线管理

**页面布局**：左右分栏
- 左侧：生产线列表
- 右侧：选中产线的工序列表

| 字段 | 说明 |
|------|------|
| 产线编码 | 唯一标识 |
| 产线名称 | 产线名称 |
| 位置 | 所在空间位置 |
| 小时产能 | 每小时产能 |

---

#### 3.9.3 生产单元-工序管理

生产线下的工序配置。

| 字段 | 说明 |
|------|------|
| 工序编码 | 唯一标识 |
| 工序名称 | 工序名称 |
| 顺序 | 在产线中的顺序 |
| 标准工时 | 单位产品的标准工时 |
| 标准人数 | 该工序的标准人数 |

---

#### 3.9.4 班次日历

定义工作班次，用于能耗分班统计。

| 字段 | 说明 |
|------|------|
| 班次名称 | 如"早班"、"中班"、"夜班" |
| 班次编码 | 唯一标识 |
| 开始时间 | 班次开始时间 |
| 结束时间 | 班次结束时间 |
| 适用星期 | 周一至周日的适用情况 |

   ├── 定义用能对象

   └── 用于能源平衡分析

## 五、常见问题（FAQ）

### Q1：系统能源类型可以删除吗？

**A**：系统预设的能源类型不能删除，但可以禁用。禁用后该类型不会出现在计量表、用能对象等的选择列表中。自定义的能源类型可以删除（如果未被引用）。

---

### Q2：计量表如何关联 IoT 设备？

**A**：
1. 确保设备已在 Xlink 平台注册
2. 编辑计量表，在「关联设备」字段填写 Xlink 设备 ID
3. 保存后系统会验证设备连接状态
4. 配置采集策略后即可自动采集数据

---

### Q3：采集策略停用后会怎样？

**A**：
- 关联该策略的计量表将停止自动采集
- 历史数据保留，不受影响
- 重新启用策略后恢复采集
- 停用期间的数据空缺需要手动补录

---

### Q4：一个计量表可以绑定多个用能对象吗？

**A**：可以。通过设置分成比例来分摊能耗：
- 在不同用能对象中绑定同一个计量表
- 为每个绑定设置分成比例（如 A 对象 70%，B 对象 30%）
- 系统按比例分摊该表的能耗到各对象

---

### Q5：基线计算需要多少历史数据？

**A**：建议至少有 30 天的历史能耗数据再配置基线：
- 固定基线：需要覆盖参考期间的完整数据
- 滚动基线：需要 N 期历史数据（N 为滚动周期）
- 回归基线：建议 3 个月以上数据以保证模型准确性

---

### Q6：用能对象的类型选错了可以修改吗？

**A**：用能对象创建后类型不可修改。如需更改类型：
1. 记录当前对象的表计绑定信息
2. 删除当前用能对象
3. 以正确类型重新创建
4. 重新绑定计量表

---

## 六、附录

### 术语表

| 术语 | 说明 |
|------|------|
| 折标煤系数 | 将各种能源折算成标准煤的换算系数，用于统一计量 |
| CO₂排放因子 | 单位能源消耗产生的二氧化碳排放量（kgCO₂/单位） |
| 采集策略 | 定义计量表自动采集数据的规则和频率 |
| 用能对象 | 消耗能源的业务实体，如设备、区域、产线 |
| 能源源头 | 能源的输入来源，如电网购电、自发电、燃料采购 |
| 能流节点 | 能源流动网络中的节点，如输入点、转换点、消耗点 |
| 基线 | 能效评估的参照基准值，用于对比分析 |
| 峰谷时段 | 按用电高峰程度划分的电价时段（尖峰、峰、平、谷） |
| EnPI | 能源绩效指标（Energy Performance Indicator） |
| ISO 50006 | 能源基线与能源绩效指标测量标准 |
| ISO 50015 | 能源绩效改进测量与验证标准 |
