# 能耗分析模块 - 用户手册

> 版本：v1.0 | 更新日期：2026-01

---

## 一、模块概述

### 1.1 能耗分析是什么

能耗分析模块基于实时采集的能耗数据，提供多维度的分析、对比、趋势、预测和报表功能，帮助企业全面掌握能源使用状况，发现节能机会，优化能源管理策略。

### 1.2 核心功能模块

| 功能模块 | 说明 |
|----------|------|
| 能源概览 | 一站式能耗监控仪表板，KPI 卡片与可视化图表 |
| 能耗趋势分析 | 多维度能耗趋势监测与同环比分析 |
| 分项能耗分析 | 按能源类型、峰谷平时段的详细分解 |
| 区域能耗分析 | 按空间/区域统计能耗分布与能源强度 |
| 产品单耗分析 | 基于实际产量的能源单位消耗分析 |
| 能源成本分析 | 能源成本结构、趋势与节能潜力评估 |
| 能效分析 | KPI 指标分析、多维钻取、对比分析 |
| 能效对标 | 行业对标、区域对标、设备对标、历史对标 |
| 能源预测 | 基于历史数据的用能预测与实际对比 |
| 能耗报表 | 区域能耗报表、月报、能流报表、每日计量报表 |

### 1.3 用户角色与常用功能

| 角色 | 主要职责 | 常用功能 |
|------|----------|----------|
| 能源管理员 | 全面能源分析与优化 | 全部功能 |
| 运维工程师 | 监控能耗异常、设备能效 | 能耗趋势、区域分析、对标 |
| 生产主管 | 产品能耗管控 | 产品单耗、成本分析 |
| 管理层 | 了解能源使用概况 | 能源概览、报表 |

## 二、快速入门（新用户必读）

### 场景 1：查看当日/本月能耗概况

1. 进入「能源管理」→「能源概览」页面
2. 选择时间范围（按日/按月）
3. 选择能源类型（默认电力）
4. 查看 KPI 卡片：本期总能耗、环比变化、同比变化、能源费用
5. 点击卡片可下钻到详细分析页面

### 场景 2：分析某区域能耗异常

1. 从「能源概览」的区域排名点击异常区域
2. 进入「区域能耗分析」页面
3. 切换到「层级视图」查看空间层级能耗分布
4. 切换到「趋势视图」查看该区域历史趋势
5. 对比「对标视图」识别是否超出基准

### 场景 3：评估产品能耗效率

1. 进入「产品单耗分析」页面
2. 选择时间周期（近 3/6/12 个月）
3. 查看产品单耗排名 TOP 10
4. 切换到「趋势视图」观察单耗变化
5. 在「对标视图」对比行业标准

## 三、功能模块详解

### 3.1 能源概览（Dashboard）

**入口**：能源管理 → 能源概览

**解决什么问题**：一站式掌握全局能源使用状况

#### 页面组成

**1. KPI 卡片**（可点击下钻）

| 指标 | 说明 |
|------|------|
| 本期总能耗 | 选定时段的能源消耗总量 |
| 环比变化 | 与上一周期对比的变化率 |
| 同比变化 | 与去年同期对比的变化率 |
| 能源费用 | 根据电价策略计算的费用 |

**2. 可视化图表**

| 图表 | 说明 |
|------|------|
| 能耗趋势图 | 多能源类型消耗变化折线图，支持显示上期对比 |
| 能源占比 | 各类能源消耗分布饼图 |
| 分项用能 | 能源消耗堆叠柱状图 |
| 区域能耗排名 | 消耗量最大的区域 TOP 10，可点击下钻 |

**筛选功能**：日期范围、时间粒度（小时/日/月）、能源类型

**导出**：支持导出 Excel

---

### 3.2 能耗趋势分析

**入口**：能源管理 → 能耗趋势

**解决什么问题**：多维度监测能耗变化，识别异常和规律

#### 页面组成

**1. 统计卡片**（6 个）

| 指标 | 说明 |
|------|------|
| 本期总能耗 | 选定时段累计能耗 |
| 环比变化 | 与上期对比 |
| 同比变化 | 与去年同期对比 |
| 日均能耗 | 平均每日消耗量 |
| 峰值能耗 | 最高单日消耗及日期 |
| 谷值能耗 | 最低单日消耗及日期 |

**2. 分析图表**

| 图表 | 说明 |
|------|------|
| 趋势图表 | 折线图展示能耗变化，支持均值参考线 |
| 异常检测气泡图 | 识别偏离均值的异常点 |
| 时段效率分布 | 按能效等级分类各时段 |
| 对标分析 | 当前值与目标值对比 |
| 详细数据表 | 支持排序、分页、迷你趋势线 |

**筛选功能**：日期范围、粒度（日/周/月）、能源类型、区域筛选

**Tab 切换**：图表视图 / 明细视图

**导出**：支持导出 Excel/CSV

| Tab | 内容 |

**导出**：支持导出 Excel/CSV

### 3.4 区域能耗分析

**入口**：能源管理 → 区域能耗

**解决什么问题**：按空间/区域统计能耗分布，识别高耗区域

#### 页面组成

**1. 统计卡片**

| 指标 | 说明 |
|------|------|
| 监测空间数 | 参与统计的空间/区域数量 |
| 总能耗 | 含同环比变化 |
| 平均能源强度 | 单位面积能耗（kWh/m²） |

**2. 五大视图 Tab**

| Tab | 功能 |
|-----|------|
| 图表视图 | 能耗排行 TOP 10 条形图、能耗分布饼图、设施类型分布、能源强度气泡图 |
| 表格视图 | 用能对象列表，支持排序（能耗/环比/同比/能源强度/占比） |

| 趋势视图 | 选择多个空间对比能耗趋势 |

**入口**：能源管理 → 产品单耗

**解决什么问题**：基于实际产量计算产品能源单位消耗

**前置条件**：已配置产品、用能对象绑定计量表

#### 页面组成

**1. 统计卡片**

| 指标 | 说明 |
|------|------|
| 监测产品数 | 参与统计的产品数量 |
| 总产量 | 累计产量 |
| 总能耗 | 累计能耗 |
| 平均单耗 | 总能耗/总产量 |

**2. 四大视图 Tab**

| Tab | 功能 |
|-----|------|
| 图表视图 | 产品单耗排名 TOP 10、产品能耗分布饼图、产品产量对比、效率分布 |
| 趋势视图 | 选择多个产品对比单耗趋势 |

| 指标 | 说明 |

### 3.10 能耗报表

**入口**：能源管理 → 能源报表

**解决什么问题**：生成标准化能耗报表，支持自定义模板

#### 报表类型

| 类型 | 说明 | 数据维度 |
|------|------|----------|
| 区域能耗报表 | 按区域/用能对象汇总能耗 | 空间层级 × 日期 |
| 月度能耗报表 | 月度能耗统计与同环比 | 用能对象 × 同环比 |
| 能流报表 | 能源从源头到终端的流向 | 节点 × 日期 |
| 每日计量报表 | 计量表日读数明细 | 计量表 × 日期 |

#### 功能操作

| 操作 | 说明 |
|------|------|
| 选择报表 | 选择报表类型和日期范围 |
| 预览报表 | 全屏预览/打印（横向打印） |
| 导出 Excel | 支持自定义模板 |
| 模板管理 | 上传/下载/删除模板 |

#### 特殊规则

- 月报仅支持同月范围，跨月自动纠正
- 能流报表需选择节点
- 每日计量报表默认显示前 50 个计量表

## 四、数据导出与报表

### 通用导出功能

- 大多数分析页面支持 Excel/CSV 导出
- 导出内容包含当前筛选结果
- 文件名格式：`{分析类型}_{日期}.xlsx`

### 自定义报表模板

**模板占位符**：

| 占位符 | 说明 |
|--------|------|
| `#HEADER_START#` | 表头开始位置 |
| `#HEADER_END#` | 表头结束位置 |
| `#DATA_START#` | 数据开始位置 |
| `#DATA_END#` | 数据结束位置 |

**使用步骤**：

1. 下载默认模板
2. 自定义样式和布局
3. 保留占位符位置
4. 上传自定义模板
5. 导出时选择自定义模板

## 五、常见问题（FAQ）

| 问题 | 解答 |
|------|------|
| 能耗趋势图中数据缺失怎么办？ | 检查计量表采集策略是否正常，或使用「能耗重算」功能补全数据 |
| 产品单耗显示异常高/低？ | 检查用能对象是否正确绑定计量表，确认产量数据是否录入 |
| 对标分析没有数据？ | 需要先在「对标设置」中配置行业基准值 |
| 如何查看某个设备的能耗？ | 在「区域能耗分析」中切换到「表格视图」，筛选到具体用能对象 |
| 环比/同比数据为什么是空？ | 需要有对应历史周期的数据才能计算同环比 |
| 预测准确率低怎么办？ | 确保历史数据充足（建议至少 30 天），并选择合适的预测方法 |
| KPI 应用下拉为空？ | 需要先在「能源管理」→「KPI 配置」中创建 KPI 应用 |
| 报表导出格式不正确？ | 检查自定义模板的占位符是否完整，或使用默认模板 |

## 六、附录

### 术语表

| 术语 | 说明 |
|------|------|
| 环比 | 与上一个相同周期（如上月）对比的变化率 |
| 同比 | 与去年同一周期对比的变化率 |
| 能源强度 | 单位面积的能耗量（kWh/m²） |
| 产品单耗 | 生产单位产品消耗的能源量 |
| 标准煤 | 能源折算的统一计量单位，1 kWh ≈ 0.1229 kgce |
| 峰谷平 | 按用电高峰程度划分的电价时段 |
| 对标 | 与基准值（行业标准、历史数据等）进行对比分析 |
| KPI 应用 | 已配置的能效指标计算规则 |
| 钻取 | 从汇总数据逐级深入到明细数据的分析方式 |

### 能效等级说明

| 等级 | 颜色 | 说明 |
|------|------|------|
| 优秀 | 绿色 | 能效处于领先水平（前 20%） |
| 良好 | 蓝色 | 能效表现良好（20%-40%） |
| 一般 | 灰色 | 能效表现一般（40%-60%） |
| 警告 | 黄色 | 能效偏低，需关注（60%-80%） |
| 差 | 红色 | 能效较差，需改进（后 20%） |

### 移动端支持

| 页面 | 移动端支持 |
|------|------------|
| 能源概览 | ✅ 简化版 KPI 卡片 |
| 区域能耗分析 | ✅ 部分视图 |
| 其他分析页面 | ❌ 建议 PC 端操作 |
