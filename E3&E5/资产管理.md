# 资产管理用户手册

> 本手册适用于资产管理员、盘点员、设备维护员和部门主管，帮助您快速掌握资产管理模块的核心功能。

---

## 一、模块概述

### 1.1 什么是资产管理？

资产管理模块帮助企业实现设备、设施、仪器等固定资产的全生命周期数字化管理，涵盖从采购入库到报废处置的完整流程。

### 1.2 核心价值

| 价值 | 说明 |
|------|------|
| 📋 资产台账数字化 | 建立完整的资产信息库，告别纸质台账 |
| 🔄 生命周期追踪 | 从采购到报废的全流程状态记录 |
| 📊 盘点数字化 | 扫码盘点，提高效率和准确性 |
| ⏰ 维护预警 | 保修、维保到期自动提醒 |
| 🔗 IoT 集成 | 关联物联网设备，实时监控资产状态 |

### 1.3 用户角色

| 角色 | 主要职责 | 常用功能 |
|------|----------|----------|
| 资产管理员 | 建立和维护资产台账 | 资产台账、类别管理、批量导入 |
| 盘点员 | 执行资产盘点任务 | 资产盘点、移动端扫码 |
| 设备维护员 | 资产维护保养 | 资产详情、维保记录 |
| 部门主管 | 资产使用监督 | 资产概览、统计分析 |

## 二、快速入门

### 场景 1：首次建立资产台账

**目标**：从零开始建立企业资产信息库

**操作步骤**：

1. **创建分类体系**
   - 导航至「资产管理」→「资产类别」
   - 点击「新建分类」，输入编码、名称
   - 可创建多级子分类（如：机电设备 > 空调设备 > 中央空调）
   - 💡 技巧：点击「导入模板」可一键导入行业标准分类
2. **建立空间体系**
   - 导航至「系统设置」→「空间管理」
   - 创建建筑、楼层、房间等空间层级
   - 用于标记资产安装位置

3. **添加资产**
   - 导航至「资产管理」→「资产台账」
   - 点击「新建资产」
   - 填写基本信息：名称、类别、制造商、型号等
   - 选择安装位置（空间）
   - 填写采购信息和保修信息（可选）
   - 点击「保存」

### 场景 2：批量导入资产

**目标**：通过 Excel 快速导入大量资产数据

**操作步骤**：

1. 进入「资产台账」页面
2. 点击右上角「批量导入」
3. 点击「下载模板」获取标准模板
4. 按模板格式填写资产信息：
   - **必填**：资产名称
   - **选填**：分类名称、位置、制造商、型号、序列号、状态、采购日期、采购价格、保修日期、备注
5. 上传填写好的 Excel 文件
6. 系统自动验证数据，显示校验结果
7. 确认无误后点击「导入」
8. ✅ 导入成功，资产自动生成编码（格式：AS-YYYYMMDD-NNN）

### 场景 3：执行资产盘点

**目标**：核查资产账实一致性

**操作步骤**：

1. **创建盘点任务**
   - 导航至「资产管理」→「资产盘点」
   - 点击「新建盘点」
   - 填写盘点名称、计划时间
   - 选择盘点范围（全部资产 / 按类别 / 按位置）
   - 指定盘点负责人和执行人
   - 点击「创建」

2. **开始盘点**
   - 在盘点任务详情页点击「开始盘点」
   - 系统自动生成待盘点资产清单

3. **逐一盘点**
   - 移动端：扫描资产二维码自动定位
   - PC端：在列表中搜索资产
   - 标记盘点结果：
     - ✅ 账实相符：资产存在且状态正常
     - ❌ 盘亏：账上有但找不到实物
     - ➕ 盘盈：发现账外资产

4. **完成盘点**
   - 所有资产盘点完成后，点击「完成盘点」
   - 系统自动生成盘点报告和差异明细

## 三、功能模块详解

### 3.1 资产台账

资产台账是资产管理的核心功能，集中管理所有资产信息。

#### 页面布局

| 区域 | 功能 |
|------|------|
| 顶部搜索栏 | 按资产名称、编码、序列号快速搜索 |
| 状态筛选 | 按资产状态筛选（全部/启用中/维护中/停用等） |
| 类别筛选 | 按资产分类筛选 |
| 操作按钮 | 资产类别、新建资产、批量导入、批量导出、批量二维码 |
| 资产列表 | 显示资产编码、名称、类别、状态、位置、保修状态等 |

#### 资产状态说明

| 状态 | 说明 | 颜色 |
|------|------|------|
| 待入库 | 新采购尚未验收启用 | 🟡 黄色 |
| 启用中 | 正常使用中 | 🟢 绿色 |
| 维护中 | 正在进行维修保养 | 🔵 蓝色 |
| 停用 | 暂时停止使用 | ⚪ 灰色 |
| 报废 | 已达报废条件 | 🔴 红色 |
| 已清理 | 物理处置完成 | ⚫ 深灰 |

#### 关键操作

| 操作 | 说明 |
|------|------|
| 点击资产行 | 进入资产详情页 |
| 二维码图标 | 生成单个资产二维码 |
| 批量导入 | Excel/CSV 批量导入资产 |
| 批量导出 | 导出当前筛选结果为 Excel |
| 批量二维码 | 批量生成选中资产的二维码（ZIP 下载） |

### 3.2 资产详情

点击任意资产进入详情页，查看完整信息。

#### 基本信息区

| 字段 | 说明 |
|------|------|
| 资产编码 | 系统自动生成，唯一标识 |
| 资产名称 | 资产显示名称 |
| 资产分类 | 所属类别（树形路径） |
| 当前状态 | 生命周期状态 |
| 制造商/型号/序列号 | 设备规格信息 |
| 采购日期/价格 | 采购信息 |
| MTBF / MTTR | 可靠性指标（如有历史数据） |

#### 位置与关联

| 字段 | 说明 |
|------|------|
| 空间位置 | 资产安装的空间层级路径 |
| 关联设备 | 绑定的 IoT 设备（Xlink 设备 ID） |

#### 功能标签页

| 标签页 | 功能 | 显示条件 |
|--------|------|----------|
| 实时状态 | IoT 设备实时数据监控 | 已关联 Xlink 设备 |
| 保修与维保 | 保修/维保服务商、到期日期 | 有保修或维保信息 |
| 附件管理 | 上传/下载资产相关文档（说明书、合同等） | 始终显示 |
| 生命周期 | 状态变更历史时间线 | 始终显示 |
| 巡检记录 | 该资产的巡检历史记录 | 始终显示 |
| 维保记录 | 相关维护保养工单 | 始终显示 |
| 相关问题 | 巡检发现的问题记录 | 始终显示 |
| 预测分析 | AI 预测性维护建议 | 始终显示 |

#### 操作按钮

- **二维码**：生成并下载资产二维码
- **编辑**：修改资产信息
- **删除**：软删除资产（可从回收站恢复）

### 3.4 资产盘点

定期核查资产账实一致性，发现和处理资产异常。

#### 盘点任务状态

| 状态 | 说明 |
|------|------|
| 计划中 | 已创建任务，等待开始 |
| 进行中 | 正在执行盘点 |
| 已完成 | 盘点结束，已生成报告 |
| 已取消 | 任务被取消 |

#### 盘点流程

```
创建任务 → 开始盘点 → 逐一核实 → 标记结果 → 完成盘点 → 生成报告
```

#### 盘点结果类型

| 结果 | 说明 |
|------|------|
| ⏳ 待盘点 | 尚未核实 |
| ✅ 账实相符 | 资产存在且信息一致 |
| ❌ 盘亏 | 账上有记录但找不到实物 |
| ➕ 盘盈 | 发现账外资产 |

#### 盘点详情页

| 区域 | 内容 |
|------|------|
| KPI 卡片 | 盘点进度、总资产数、已盘点数、盘亏数、盘盈数 |
| 差异明细 | 快速定位盘亏/盘盈资产 |
| 盘点列表 | 所有待盘点资产及结果 |

---

### 3.5 资产概览（仪表盘）

快速了解资产整体状况，及时发现问题。

#### 统计卡片

| 指标 | 说明 |
|------|------|

| 使用中 | 启用中状态的资产数 |
| 维护中 | 正在维护的资产数 |
| 资产总值 | 所有资产采购价格汇总 |

#### 其他模块

| 模块 | 内容 |
|------|------|
| 维护提醒 | 保修/维保即将到期或已逾期的资产 |
| 盘点进度 | 进行中盘点任务的完成情况 |

## 五、生命周期管理

### 资产状态流转

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   采购入库        验收启用         正常使用                              │
│   (pending) ───→ (active) ───────────────────────────────┐              │
│                      │                                    │              │
│                      ↓                                    │              │
│              ┌── 维护保养 ──┐                             │              │
│              │ (maintenance)│                             │              │
│              └──────────────┘                             │              │
│                      │                                    │              │
│                      ↓                                    ↓              │
│                 恢复使用 ←───────────────────────── 停用(inactive)       │
│                 (active)                                  │              │
│                                                           ↓              │
│                                                     报废(scrapped)       │
│                                                           │              │
│                                                           ↓              │
│                                                   物理处置(disposed)      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 生命周期记录

每次状态变更自动记录到「生命周期」标签页：

| 记录内容 | 说明 |
|----------|------|
| 变更时间 | 状态变更发生的时间 |
| 原状态 | 变更前的状态 |
| 新状态 | 变更后的状态 |
| 变更原因 | 状态变更的原因说明 |
| 操作人 | 执行变更的用户 |
| 审批状态 | 是否需要审批及审批结果 |

## 六、常见问题（FAQ）

| 问题 | 解答 |
|------|------|
| 资产编码可以修改吗？ | 创建后不可修改，确保唯一性和追溯性。如需更换编码，需删除后重新创建。 |
| 删除的资产可以恢复吗？ | 是的，系统采用软删除机制，管理员可从回收站恢复已删除资产。 |
| 如何关联 IoT 设备？ | 编辑资产时，在「关联设备」字段选择对应的 Xlink 设备 ID。关联后可在资产详情查看实时数据。 |
| 盘点时扫不到二维码怎么办？ | 可在盘点列表中手动搜索资产名称或编码进行盘点。 |
| 如何批量更新资产类别？ | 目前需逐个编辑资产的类别。如需批量变更，可导出数据修改后重新导入（会创建新资产）。 |
| 保修到期如何提醒？ | 系统自动在「资产概览」的维护提醒模块显示即将到期和已逾期的资产。 |
| 导入时类别匹配不上怎么办？ | 请确保 Excel 中的分类名称与系统中完全一致（包括大小写和空格）。可先导出现有分类作为参考。 |
| 如何查看资产的巡检和维修历史？ | 进入资产详情页，切换到「巡检记录」或「维保记录」标签页查看。 |
